<Update label="2024-10-15" description="v1.2.0" tags={["feature", "improvement"]}>

## New Features

- Added `<Columns>` component support for responsive grids up to 5 columns.
- Introduced `<Video>` and `<Iframe>` embeds with customizable dimensions.

## Improvements

- Enhanced Mermaid diagram rendering with better error handling for complex graphs.
- Optimized parsing speed for pages with nested JSX components by 40%.

## Bug Fixes

- Fixed line number display in `<CodeGroup>` when `show-lines="true"`.
- Resolved escaping issues for `{` and `}` in prose text.

</Update>

<Update label="2024-09-20" description="v1.1.0" tags={["feature", "bugfix"]}>

## New Features

- New `<ParamField>` and `<ResponseField>` for API documentation sidebars.
- Support for `tags` arrays in `<Update>` components with color-coded badges.

## Bug Fixes

- Corrected attribute parsing for kebab-case props like `default-open`.
- Fixed nested code block rendering using 4 backticks properly.

<Update label="2024-08-10" description="v1.0.0" tags={["feature", "breaking"]}>

## New Features

- Initial release with core MDX components: `<Callout>`, `<Card>`, `<Steps>`, and more.
- Full YAML frontmatter validation with no-blank-line enforcement.

## Breaking Changes

- Removed H1 support in body content; always start with H2.
- Standardized all attributes to double quotes only.

## Improvements

- Added Lucide icon integration with 100+ available icons.

</Update>
